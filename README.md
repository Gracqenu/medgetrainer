# Mirror's Edge Trainer Mod

## Setup 
1. Run the launcher either before Mirror's Edge or when Mirror's Edge is running.
2. Press `Insert` to view the in-game menu where you can adjust settings as you please.
3. Done
## Features
- Trainer
    - Save/load state
    - God mode
    - Fly mode
    - KG tool
    - Beamer tool
    - Strang tool
    - Level Streamer
